Задание по карте №2

  • Реализовать карту через API Яндекс карт;

  • Настроить кастомный пин на карте так, что бы его стрелочка четко указывала на центр координат, которые вы можете придумать сами;

  • На десктопной версии при скролле колесиком мышки, карта не должна масштабироваться;

  • Создать и расположить пины на карте из json файла;

  • При клике на каждый пин выводить какую нибудь информацию в небольшом всплывающем окошке, которое будет появляться над пином;

Для изменения масшата карты используйте CTRL + колесико мышки
